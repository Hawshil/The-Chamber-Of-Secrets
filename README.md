# The Chamber Of Secrets

A first-person stealth-puzzle dungeon experience built in Unreal Engine 5.

The player infiltrates an ancient underground dungeon to steal a mysterious idol hidden behind a secret chamber. To reach it, the player must explore the environment, uncover hidden mechanisms, manipulate objects, and solve environmental puzzles using physics and observation.

This project is primarily focused on learning and implementing core gameplay programming and level design systems in Unreal Engine.

---

## Core Gameplay Mechanics

- Object holding and grabbing system
- Physics interaction
- Weight-based pressure buttons
- Hidden / secret doors
- Lumen-based dynamic lighting exploration
- Environmental puzzle interactions
- Exploration-driven progression

---

## Technical Learning Goals

This project is being developed as a hands-on learning experience for gameplay systems and Unreal Engine architecture.

### Gameplay Programming
- Line tracing
- Collision handling
- Interaction systems
- Actor Components
- Scene Components
- Physics-based interactions
- Blueprint communication
- Gameplay state logic

### Level Design
- Modular level design
- Dungeon layout planning
- Environmental storytelling
- Lighting composition
- Puzzle space design
- Player guidance and navigation
